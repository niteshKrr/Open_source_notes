# Operators & Keywords

## 1. Operators

Operators perform operations on variables/values.

1. Arithmetic Operators
    ```python
    +  -  *  /  %  **
    ```

2. Comparison Operators
    ```python
    ==  !=  >  <  >=  <=
    ```

3. Logical Operators
    ```python
    and or not
    ```

4. Assignment Operators
    ```python
    =  +=  -=  *=  /=
    ```

5. Bitwise Operators
    ```python
    &  |  ^  ~  <<  >>
    ```

6. Membership Operators
    ```python
    in, not in
    ```

7. Identity Operators
    ```python
    is, is not
    ```
